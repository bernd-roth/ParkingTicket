# ParkingTicket

Book your parking ticket in at least whole Austria without any intervention and furthermore never forget to book it again.

## Description

ParkingTicket is used for automatically book tickets via SMS.

##  Functions

1. Sending alternate parking tickets
2. Never forget to book parking tickets because it will be done automatically now for you
3. Receive a spoken message that your parking ticket will end in a few minutes
4. Get an overview of where parksheriffs are currently underway
5. Get an overview of where are parkig zones, park garages, and many more things
6. Send a parking ticket automatically again, if because of an cell network outage the app was not able to send it
